java 17 Stream流新api总结

Heartape 2022-07-07 stream

# 1.toList

类似于原来的collect(Collectors.toList())，但是有些许不同，

  • toList生成的list为不可变，改变后会产生异常
  • toList直接使用的toArray方法，而Collectors.toList()则利用forEach循环取值性能较差

# 2.takeWhile,dropWhile

takeWhile:获取满足条件的元素，遇到第一个不满足条件的就停止 dropWhile:丢弃满足条件的元素，遇到第一个不满足条件的就停止

# 3.mapMulti

类似于flatMap，但是底层有较大的优化，无需创建新的stream流，减小了开销，而且语法也更加优雅

import java.util.List;

class Test {
    /**
     * 将所有元素复制一份
     */
    public List<Interger> copy(List<Interger> list) {
        return list.stream().<Integer>mapMulti((value, consumer) -> {
            consumer.accept(value);
            consumer.accept(value);
        }).toList();
    }
}
