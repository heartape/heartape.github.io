# 1.Lambda

为了加入Lambda和增强接口的扩展性,接口可以有默认方法,默认方法拥有具体实现,在方法加前缀default;

# 1.函数式接口规范

1.注解@FunctionalInterface,只起声明的作用;

2.只能有一个未实现的方法,其余为default默认方法;

3.Lambda表达式引用的变量不能改变

@FunctionalInterface
public interface Eat {
    String eat(String food);
}

public class Person {
    public static String eat(Eat eat, String food) {
        return eat.eat(food);
    }

    public static String eat(String food) {
        return "eat:" + food;
    }

    public static void main(String[] args) {
        // 三者效果一致，但方式3更优雅，并且将实现分离便于扩展
        Eat eat = new Eat() {
            @Override
            public String eat(String food) {
                return "eat:" + food;
            }
        };
        String eat1= Person.eat(eat, "汉堡");
        String eat2 = Person.eat(food -> "eat:" + food, "汉堡");
        String eat3= Person.eat(Person::eat, "汉堡");
    }
}

# 2.stream

# 1.串行流

public class Test {
    public void test1(){
        List<Integer> list = Arrays.asList(1, 2, 3, 4, 5, 6, 7, 8, 9);
        list
                //流
                .stream()
                //映射(参数函数型)
                .map(item -> item*10)
                //过滤(参数断言型)
                .filter(item -> item>50)
                //比较()参数比较器
                .sorted((a,b) -> a-b)
                //遍历(参数消费型)
                // .forEach(item -> System.out.println(item))
                //累加
                .reduce();
    }

    public void test2(){
        List<Person> list = Arrays.asList(
                new Person("张三",18),
                new Person("大爷",58),
                new Person("张四",58));

        // 排序,传入一个比较器
        list.sort(Comparator.comparingInt(Person::getAge));
        // 流排序,传入一个比较器
        list.stream().sorted(Comparator.comparingInt(Person::getAge));
        // reversed()倒序
        list.sort(Comparator.comparingInt(Person::getAge).reversed());
        // 多重排序
        list.sort(Comparator.comparingInt(Person::getAge).reversed().thenComparing(Person::getName));
    }
}
