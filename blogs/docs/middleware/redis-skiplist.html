# 1.skip list介绍

跳跃表(skip list)由William Pugh在论文《Skip lists: a probabilistic alternative to balanced trees》中提出。 其时间复杂度平均O(logN)、最坏O(n)，效率与平衡树相当，重要的是不需要复杂的平衡操作。 skip list示意图如下:

start  ->   head   ->    2      ->      5
              ↓          ↓              ↓
            head -> 1 -> 2 -> 3   ->    5
              ↓     ↓    ↓    ↓         ↓
            head -> 1 -> 2 -> 3 -> 4 -> 5

# 2.实现分析

# 1.查询

如上图所示，查询时从左上往右下，直到寻找到目标值。 因为skip list是有序的，本质上就是通过空间换时间的方式实现近似的二分查找，每次往下一层都可以排除近似一半的数据。 我这里就写个伪代码

public class Test {
    public String select(int target) {
        Node current = start();
        for (;;) {
            if (current == null)
                return "error";
            Node right = current.right;
            int value = right.value;
            if (value > target) {
                current = current.down;
            } else if (value < target) {
                current = current.right;
            } else {
                return current.content;
            }
        }
    }
}

# 2.插入

插入流程与查询基本一致，只是增加了链表的插入和超出高度后扩容。 同样写个伪代码

public class Test {
    public void insert(int target) {
        List<Node> list = new ArrayList<>();
        Node current = start();
        int maxLevel = level();
        for (;;) {
            if (current == null)
                return "error";
            Node right = current.right;
            int value = right.value;
            if (value >= target) {
                if (right.level == 0) {
                    // 随机高度
                    int level = random();
                    if (level > maxLevel)
                        // 扩容
                        addLength(level);
                    // 执行插入（单向链表的伪代码，详细的就不写了）
                    // 从list里筛选出符合level的节点
                    // current之后创建节点，value = target
                    // 将新节点插入到current和current.next之间
                    process(list, level, current, target);
                    return;
                } else {
                    list.add(current);
                    current = current.down;
                }
            } else {
                current = current.right;
            }
        }
    }
}
