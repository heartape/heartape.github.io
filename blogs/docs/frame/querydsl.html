querydsl简单入门

Heartape 2022-11-05 querydsl

# 1.起因

通常开发难免会写大量的crud，为了提高效率，大家往往会选择mybatis-plus来免去手写单表查询sql的时间。 此前作者也一直在使用mybatis-plus，但是这个使用越久越觉得框架本身缺陷较大，设计不够优雅，便找到了querydsl。

# 2.简介

Querydsl是一个支持构建静态类型SQL查询的框架。可以通过像Querydsl这样的流畅API来构造查询，而不是将查询写入内联字符串或将其外部化为XML文件。

/**
 * 复杂查询
 */
@Service
public class UserServiceImpl implements IUserService {
    @Autowired
    private JPAQueryFactory queryFactory;

    @Override
    public User getOne(Integer id) {
        QUser user = QUser.user;
        return queryFactory
                .select(user)
                .from(user)
                .where(user.id.eq(id))
                .fetchOne();
    }
}
