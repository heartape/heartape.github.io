git基础命令

Heartape 2022-06-12 git

# 1.基本命令

# 1.配置

  • 用户名:$ git config --global user.name + 用户名
  • 邮箱:$ git config --global user.email + 邮箱
  • 配置信息:$ git config --list

# 2.提交

初始化仓库

git init

查看文件状态

git status

文件列表,追踪文件

git add

提交信息,向仓库中提交代码,m后面为自定义提交信息

git commit -m first

查看提交记录

git log

# 3.撤销

用暂存区的文件覆盖工作目录中的文件

git checkout + 文件

将文件从暂存区删除

git rm --cached + 文件

将仓库中指定的更新记录回复出来,并覆盖暂存区和工作目录,仓库中指定的更新记录删除

git reset --hard + commitId

# 2.分支

  • 主分支
  • 开发分支
  • 功能分支

# 1.分支命令

查看分支

git branch

创建分支

git branch +分支名称

切换分支

git checkout +分支名称

创建并切换分支

git checkout -b +分支名称

合并分支,要切换到合并后的分支上

git merge +来源分支

删除分支,合并后才允许删除

git branch -d +分支名称

强制删除

git branch -D +分支名称

# 2.暂时保存更改

将未完成的工作暂时存储到剪切板上,以便切换分支,因为切换分支需要提交,但工作未完成不想提交. 存储临时改动

git stash

恢复改动

git stash pop

# 3.远程仓库

远程仓库我一般直接在页面上创建后直接拉取后使用

获得ssh公钥

ssh-keygen -t rsa -C "1343348876@qq.com"

公钥添加到码云,测试添加是否成功

ssh -T git@gitee.com

建立远程仓库

git push + 地址 + 分支

建立远程仓库,并且提交时只需要git push

git push -u +地址 +分支

为远端仓库建立别名

git remote add +别名 +地址

向远端仓库推送

git push +别名 +分支

克隆仓库

git clone +地址
