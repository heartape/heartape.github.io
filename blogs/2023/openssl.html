openssl

Heartape 2023-01-30 openssl

# 1.命令

安装：

  • yum install openssl openssl-devel 生成：
  • openssl genrsa -out rsa_private_key.pem 2048
  • openssl pkcs8 -topk8 -inform PEM -in rsa_private_key.pem -outform PEM -nocrypt -out rsa_private_key_pkcs8.pem
  • openssl rsa -in rsa_private_key.pem -pubout -out rsa_public_key_pkcs8.pem

# 2.说明

  • 不同的语言会使用不同格式的密钥。java使用的是pkcs8格式，如果需要在java中使用，则需要使用pkcs8格式的公私钥。
  • 生成的私钥默认为pkcs1格式，需要命令3转换
  • 生成的公钥默认为pkcs8格式
