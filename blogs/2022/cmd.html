windows cmd批处理curl

Heartape 2022-07-13 cmd

前几天有一个跑脚本发送http请求的需求，就到网上找了半天，总算是找到了一个还行的方案。 据说jq可以解析，以后试试再更新

@echo off
@chcp 65001 > nul
set success={"code":1,"message":"成功"}
@for /f %%i in ('curl -X DELETE "http://localhost/xxx/xxx"') do set result=%%i
if "%result%"=="%success%" (echo 成功) else (echo 失败)
pause
@echo off
