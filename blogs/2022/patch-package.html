patch-package nodejs依赖魔改

Heartape 2022-06-13 patch-package

在使用依赖时难免会需要修改源码，但是直接修改源码后重新拉取依赖时，又需要再次修改，比较麻烦。这时就需要使用patch-package，他以补丁的形式改变了源码。

# 1.安装

npm i patch-package --save-dev

# 2.生成补丁

npx patch-package package-name

# 3.添加启动命令

{
  "scripts": {
    "postinstall": "patch-package"
  }
}
